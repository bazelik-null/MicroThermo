# MicroThermo
Repository of my code for a thermostat based on Arduino, functioning with values from two thermometers.

You can check and edit configuration at MicroThermo/include/temperatureSensor.h

## Indicators
There is 3 type of indicators
- 5v    - Power Indicator (Yellow)
- PIN 3 - Relay Indicator (Green)
- PIN 4 - Error Indicator (Red)

## Thermometers
I recommend to use two DS18B20 thermometers
- PIN 12 - Thermometer inside
- PIN 11 - Thermometer outside
